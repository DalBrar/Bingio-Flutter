# Bingio-Flutter
A catalog app to track and browse TV Shows and Movies

## Technologies being used:
- Flutter
- Firebase

## Designed for and tested on:
- Android TV

## Setup
- Besure to create the `lib/secrets/secrets.dart` file as per the README in that folder
- Dont forget to add the Goolge Android API key to `/android/app/google-services.json`
- Then run `flutter pub get` or `menu.dat`

### Incase of accidently uploading private API keys, follow these steps:
- [Removing sensitive data from a repository](https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/removing-sensitive-data-from-a-repository)
